# Heart Disease Diagnostic Intelligence System

## Project Overview

Cardiovascular diseases are the leading cause of death globally. This project provides an end-to-end Machine Learning
solution to assist clinical decision-making by predicting the risk of heart disease based on 13 critical physiological
parameters.This repository demonstrates a full ML Life Cycle: from modular data preprocessing and hyperparameter tuning
to a production-ready web application and containerization.

## System Architecture

The project is built with a focus on modularity and reproducibility. The system is divided into three main layers:

* Data Layer: Automated pipelines for cleaning and feature scaling.
* Model Layer: Optimized classifiers with Hyperparameter Tuning (GridSearchCV).
* Application Layer: A real-time inference engine served via Streamlit.

## Key Features

* Standardized Data Pipeline: Automated StandardScaler persistence to ensure consistency between training and real-time inference.

* Model Benchmarking: Comparative analysis between Logistic Regression, Random Forest, and KNN.
 
* Hyperparameter Optimization: Used GridSearchCV with 5-fold Cross-Validation, specifically optimizing for Recall to minimize clinical False Negatives.
 
* Comprehensive Logging: Integrated professional logging system for debugging and monitoring execution flow.

* Interactive UI: A custom-built dashboard for medical practitioners to input patient data and receive instant risk assessments.
 
* Containerization: Fully Dockerized to ensure "works on my machine" consistency across all environments.

## Getting Started
**Prerequisites**
* Python 3.11+
* Docker (Optional, for containerization)

**Local Setup**
cd heart-disease-ai
* Install Dependencies:
pip install -r requirements.txt
* Run Preprocessing:
python src/data_prep.py
* Launch the App:
streamlit run src/app.py

**Running with Docker**

docker build -t heart-disease-app .
docker run -p 8501:8501 heart-disease-app
